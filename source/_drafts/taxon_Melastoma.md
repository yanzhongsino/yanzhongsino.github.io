# 1. 野牡丹属（Melastoma）简介



# 中国野牡丹属物种

## 地菍（M. dodecandrum）
1. 形态特征
- 植株矮小，，高10-60厘米以下；
- 茎匍匐上升，逐节生根，小枝披散；
- 叶片长4厘米，宽2厘米以下。
2. 花果期
- 花期5-7月，果期7-9月。
3. 分布
- 贵州、湖南、广西、广东（海南岛未发现），江西、浙江、福建
- 越南
4. 生境
- 海拔1250米以下
- 山坡矮草丛中
- 酸性土壤常见的植物

## 毛菍（M. sanguineum）
1. 形态特征
- 花大，花瓣长3-5厘米
- 果大，直径1.2厘米以上
2. 花果期
- 花果期几乎全年，通常在8-10月。
3. 分布
- 广西、广东
- 印度、马来西亚至印度尼西亚
4. 生境
- 海拔400米以下
- 坡脚、沟边，湿润的草丛或矮灌丛中

## 大野牡丹（M. imbricatum）
1. 形态特征
- 大灌木或小乔木，高1-7米
- 叶片大，长8-21厘米；
- 叶柄长，长1.8-6.5厘米。
2. 花果期
- 花期6-7月
- 果期12月至翌年2-3月。
3. 分布
- 云南东南部、广西西南部
- 印度、缅甸、中南半岛至印度尼西亚的苏门答腊
4. 生境
- 海拔140-1420米
- 密林下湿润的地方

## 展毛野牡丹（M. normale）
1. 形态特征
- 茎被平展的长粗毛及短柔毛。
2. 花果期
- 花期春至夏初（云南南部有时9-11月）
- 果期秋季（云南南部有时5-6月）
3. 分布
- 西藏、四川、福建至台湾以南各省区
- 尼泊尔、印度、缅甸、马来西亚及菲律宾等地也有，爪哇不产。
4. 生境
- 海拔150-2800米
- 开朗山坡灌草丛中或疏林下
- 酸性土常见植物

## 印度野牡丹（M. malabathricum）
1. 形态特征
- 花瓣间通常有间隙
- 小枝密被贴伏鳞片
- 叶背面密被糙伏毛和被微柔毛， 正面密被糙伏毛
2. 花果期
- 花期2-8月
- 果期7-12月
3. 分布
- 福建，广东，广西，贵州，海南，湖南，江西，四川，台湾，西藏东南部，云南，浙江
- 柬埔寨，印度，日本，老挝，马来西亚，缅甸，尼泊尔，菲律宾，泰国越南；太平洋群岛
4. 生境
- 海拔100-2800米。
- 小路边，空旷地带，草地，灌丛，灌丛，疏林，竹林。

## 野牡丹（M. candidum）
1. 形态特征
- 花瓣间无间隙；
- 叶片卵形或广卵形；
- 叶面密被糙伏毛和短柔毛，基出脉7；
- 萼片卵形或略宽。
2. 花果期
- 花期5-7月
- 果期10-12月
3. 分布
- 云南、广西、广东、福建、台湾。广东南部沿海岛屿。
- 印度支那
4. 生境
- 生于海拔约120米以下
- 山坡松林下或开朗的灌草丛中
- 酸性土常见的植物。模式标本采自广东南部沿海岛屿。

## 枝毛野牡丹（M. dendrisetosum）
1. 形态特征
- 茎密被长绢毛及微柔毛，长绢毛中部以下尚有微柔毛；
- 叶片背面密被长柔毛及微柔毛。
2. 花果期
- 花期约4月，果期未详。
3. 分布
- 海南（乐会等） 
4. 生境
- 海拔70-100米
- 疏、密林缘，阳处路旁或沟边

## 紫毛野牡丹（M. penicillatum）
1. 形态特征
- 茎密被外反的淡紫色长粗毛，毛基部略膨大；
- 叶片背面密被糙伏毛。
2. 花果期
- 花期3-4月
- 果期11月至翌年1月。
3. 分布
- 海南
- 菲律宾
4. 生境
- 海拔380-1300米
- 山坡密林下

## 野牡丹vs印度野牡丹
1. 野牡丹
- 花瓣间无间隙
2. 印度野牡丹
- 花瓣间常有间隙

## 枝毛野牡丹vs紫毛野牡丹
1. 枝毛野牡丹
- 小枝密被长绢毛及微柔毛；叶面被糙伏毛，背面密被长柔毛及微柔毛；花萼密被从基部分枝的刺毛。
2. 紫毛野牡丹
- 茎密被外反的淡紫色长粗毛，毛基部略膨大；叶片背面密被糙伏毛。
